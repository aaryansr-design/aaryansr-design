## Research & Projects

### Sentencing Audit Pipeline (RAG + Public Policy)

Designed a retrieval-augmented pipeline to analyze federal sentencing data at scale.

- Structured 60M+ variables across 60k+ cases  
- Built document parsing and OCR ingestion workflows  
- Implemented vector retrieval for legal document analysis  
- Focused on embedding audit layers into model pipelines  

**Core Question:**  
How can AI systems surface institutional disparities without amplifying them?

---

### AI Authorship Governance Framework

Developed an AI-authorship certification model within university governance structures.

- Designed human-in-the-loop verification workflows  
- Structured confidence thresholds and override mechanisms  
- Negotiated student–faculty incentive alignment  
- Translated policy into enforceable architecture  

**Insight:**  
Technical precision alone does not create trust — institutional legitimacy does.

---

### Medical Supply Classification System

Built a human-in-the-loop vision system for medical supply sorting centers.

- Image classification with confidence scoring  
- Volunteer verification interface  
- Traceable shipment metadata logging  
- Structured retraining feedback loops  

**Design Principle:**  
Accuracy is insufficient without transparency and override capacity.
